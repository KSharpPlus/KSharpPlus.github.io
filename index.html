    KSharpPlus Documentation

    KSharpPlus (K#+) is an official .NET wrapper for the Kuracord API, based off DSharpPlus, but rewritten for Kuracord.

    Getting Started

    New users will want to take a look through the articles for quick start guides, tutorials, and examples of use. Once you've gotten through the articles, head on over to the API Documentation for all classes and methods provided by this library.

    Source and Contributors

    The repository containing the source code for this library can be found here. Contributions are welcomed.
